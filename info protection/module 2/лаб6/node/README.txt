Лаба 6

==============================
Для розробки використовувався Node.JS (можна скачати на офф сайті)

index.js -файл серверу, в якому побудована логіка криптографа

папка public - файли front-end


1. При першому запускі потрібно виконати команду для встановлення необхідних модулів 
	npm i
в данній директорії (необхідно виконати лише 1 раз).

2. Для запуску сервера 
	npm test
або 
	node index.js

3. В браузері відкрити http://localhost:3010
